CSV to DSS conversion¶

In this page, I am going to show how to convert standard csv files containing distribution data into .dss files which can be used to perform distribution system analysis. Standard csv files are csv files generated using class CSVFormatter. Let’s see the description of the class you would use for for this conversion.
